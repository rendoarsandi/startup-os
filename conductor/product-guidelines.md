# Product Guidelines: AI CFO

## Design Principles
- **High-Tech & Sleek:** The interface should feel modern and sophisticated, utilizing a dark-mode first approach with subtle glowing accents to denote AI activity.
- **Data Clarity:** Financial information must be presented clearly, using intuitive visualizations and avoiding unnecessary jargon.
- **Mobile-First Design:** The application must be fully functional and highly performant on mobile devices to support on-the-go tracking.

## User Experience (UX)
- **Frictionless Interaction:** Every interaction should be optimized for speed. Minimize input steps and use AI to predict and pre-fill data whenever possible.
- **Professional & Trustworthy Tone:** All communication from the AI CFO should be concise, accurate, and professional, instilling confidence in the user.

## Technical Standards
- **Cloudflare Edge Optimized:** Every feature must be designed to run within the Cloudflare Workers environment, avoiding Node.js APIs.
- **Security First:** Financial data is sensitive. All architectural decisions must prioritize data privacy and secure authentication.
- **Performance Budget:** Maintain sub-second response times for all critical user actions.
