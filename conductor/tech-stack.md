# Technology Stack: AI CFO

## Frontend / User Interface
- **Framework:** React + Vite
- **Styling:** Tailwind CSS (v4)
- **Data Visualization:** Recharts / Chart.js

## Backend / API
- **Framework:** Hono (optimized for edge deployment)
- **Authentication:** BetterAuth
- **AI Integration:** Google AI SDK (Gemini)

## Database / Edge Infrastructure
- **Deployment Platform:** Cloudflare Workers & Pages
- **Database:** Cloudflare D1
- **ORM:** Drizzle ORM / Kysely
- **Language:** TypeScript (Strictly typed, NO Node.js APIs)
